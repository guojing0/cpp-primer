## 变量和基本类型

### 算术类型

- 当明知数值不可能为负数的时候使用无符号类型（`unsigned`）。
- 尽量使用 `int` 执行整数操作，若超过 `int` 的表示范围，则使用 `long long` 类型。
- 尽可能使用 `double` 而不是 `float` 类型。
